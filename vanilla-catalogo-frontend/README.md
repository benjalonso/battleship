## Catalogo Vanilla JS

Proyecto realizado en Javascript puro, que entrega una interfaz de usuario con un catalogo de productos, que contiene las siguiente funcionalidades: 

- Paginacion
- Carro de compras 
- Categorias 
- Filtro de busqueda desde servidor
